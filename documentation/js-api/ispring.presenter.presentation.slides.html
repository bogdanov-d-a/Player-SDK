(static) SlideType :string

The SlideType object properties provide values for the type() method of the ISlide interface.
Type: string
Properties:
Name Type Default Description
PRESENTATION_SLIDE string slide Indicates that the slide is an ordinary presentation slide.
INTERACTION_SLIDE string interaction Indicates that the slide is an iSpring Visuals interaction.
QUIZ_SLIDE string quiz Indicates that the slide is an iSpring QuizMaker quiz.
SCENARIO_SLIDE string scenario Indicates that the slide is an iSpring Visuals interaction.
