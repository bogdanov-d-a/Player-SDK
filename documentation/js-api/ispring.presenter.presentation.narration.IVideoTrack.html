playbackStateChangedEvent() → (non-null) {ispring.events.IEventDispatcher}

Returns the IEventDispatcher interface dispatching the playbackStateChanged event when the narration track changes its playbackState.

The returned event dispatcher passes the Narration Track to the listener function. See the following example.

var videoTrack = videoTracks.getVideoTrack(i);
videoTrack.playbackStateChangedEvent().addHandler(function(track) {
    if (track.playbackState() == NarrationTrackPlaybackState.ACTIVATED)
    {
        // ...
    }
});
