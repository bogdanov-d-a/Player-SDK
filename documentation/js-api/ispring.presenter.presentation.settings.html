(static, non-null) PresentationResumeMode :string

The PresentationResumeMode object properties provide values for the resumeMode() method of the IPlaybackSettings interface.
Type: string
Properties:
Name Type Default Description
PROMPT_TO_RESUME string prompt Ask the user if the presentation playback should be resumed from the last slide viewed.
ALWAYS_RESUME string always Don't ask the user, always resume the presentation playback from the last slide viewed.
NEVER_RESUME string never Don't ask the user, start the presentation from the beginning.
